# Flipkart Landing Page Clone

## 📌 Project Overview
This project is a **Flipkart landing page clone** built **entirely from scratch** using **pure HTML, CSS, and JavaScript**, without any frameworks or libraries.

I created this project as part of my frontend learning journey. The main goal was to deeply understand **web layouts, styling techniques, and animations** by implementing everything manually.

🔗 **Live Demo:** [View Website](https://shrutipandya3.github.io/Flipkart-Landing-Page-Clone/)

---

## 🎯 Purpose of the Project
- To understand **how real-world e-commerce layouts are structured**
- To implement **animations and interactions using pure CSS & JavaScript**
- To strengthen **core frontend fundamentals** before moving to frameworks

---

## 🛠️ Tech Stack
- **HTML5** – Semantic structure and layout
- **CSS3** – Flexbox, positioning, transitions, animations
- **JavaScript (Vanilla JS)** – DOM manipulation and interactions

> ❌ No frameworks  
> ❌ No libraries  
> ❌ No UI kits  

Everything was written manually to understand the fundamentals.

---

## ✨ Features
- Flipkart-style landing page UI
- Product sections and banners
- Hover effects and micro-interactions
- Mini animations using **pure CSS & JS**
- Fully hand-crafted layout and styling

---

## 🧠 Key Learnings
- Deep understanding of **CSS layout systems (Flexbox, positioning)**
- How complex UI designs are broken down into smaller components
- Writing **clean, structured HTML**
- Implementing animations without external libraries
- DOM manipulation using vanilla JavaScript
- Importance of pixel-perfect UI and spacing

This project built a strong foundation that later helped me transition smoothly into **React and the MERN stack**.
